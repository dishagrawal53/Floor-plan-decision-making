# Floor Plan Generator - Frontend

## Setup

1. Install Node.js 16+
2. Install dependencies:
```bash
   npm install
```

3. Update API URL in `src/components/FloorPlanGenerator.jsx`:
```javascript
   const API_URL = 'http://localhost:8000';
```

4. Run development server:
```bash
   npm run dev
```

5. Build for production:
```bash
   npm run build
```

## Technology Stack

- React 18
- Tailwind CSS
- Lucide React (icons)
- Vite (build tool)
